Driver to open an set an LETIMER peripheral. 

This routine is a low level driver. The application code calls this function to open one of the LETIMER peripherals for PWM operation to directly drive GPIO output pins of the device and/or create interrupts that can be used as a system \char`\"{}heart beat\char`\"{} or by a scheduler to determine whether any system functions need to be serviced.\+Clearing and enabling the registers

\begin{DoxyNote}{Note}
This function is normally called once to initialize the peripheral and the function \mbox{\hyperlink{letimer_8c_a87f7457a1824194f038e69c576dd7748}{letimer\+\_\+start()}} is called to turn-\/on or turn-\/off the LETIMER PWM operation,clearing and enabling the registers as well.
\end{DoxyNote}
